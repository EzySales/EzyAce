7‑Day Onboarding (Quick Start)

Give new agents this checklist and ask them to complete within first 7 days.

  1. Day 1 — Intro: REN roles, commission, core documents (SPA, LOA).
  2. Day 2 — Tools: WhatsApp Business, Canva kit, folder structure.
  3. Day 3 — Listing: Photo guide, caption templates, where to post.
  4. Day 4 — Prospecting: Lead sources, reply vs convert scripts.
  5. Day 5 — Viewing: Safe viewing checklist, script, objection handling.
  6. Day 6 — Closing: Booking flow, deposit, loan basics.
  7. Day 7 — Targets: Weekly KPI setup and personal action plan.

Tip: Ask each new agent to record 3 short video roleplay clips and upload to the "Training Videos" section.

Admin & Deployment (Netlify)

This site is a single-file static site ready to deploy to Netlify. You can edit the content directly in code or replace video/file links after uploading assets.

Quick Deploy Steps

  1. Create a new GitHub repo and push this index.html file.
  2. Sign in to Netlify, "New site from Git", connect the repo and deploy. Netlify will publish a URL.
  3. To update: edit index.html and push to Git; Netlify auto-deploys.

If you prefer direct upload to Netlify (ZIP)

Zip the site files and drag to Netlify "Sites" > "Add new site" > "Deploy manually".

Admin Tips

  • Store heavy videos on YouTube (unlisted) or Google Drive and paste/embed links here to keep site light.
